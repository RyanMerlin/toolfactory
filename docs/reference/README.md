# Reference Pack

This folder contains a small local reference pack for agents working on Tool Factory and Alteryx SDK integrations.

Goal:
- reduce web lookups during active work
- keep the reference set small and opinionated
- anchor implementation decisions to current official docs

Use these files as the local first stop before browsing the web.

## Maintenance Rule

- Keep summaries short.
- Keep source URLs current.
- Prefer official docs only.
- Add a reference only when Tool Factory actively uses it.
